If you get issues;

1 - The exe is unsigned, so Windows or Defender treats it as untrusted.

2 - The user downloaded it from a browser or GitHub and the file got an internet mark attached to it, which can block launch.

3 - Antivirus quarantined or partially blocked the file.

4 - The target folder permissions do not allow execution, especially in somewhere like C:/Games with restricted ACLs.

Try the following;

1 - Right-click the exe, open Properties, and see whether there is an Unblock button or checkbox.

2 - Check Windows Security, Protection history, for a Defender block or quarantine event.

3 - Copy the exe to the Desktop and run it there to rule out folder permissions.

4 - Test the exe directly.
